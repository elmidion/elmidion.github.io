GPT Information Extractor

문서에서 자동으로 정보를 추출하는 AI 기반 툴

📌 프로젝트 개요

GPT Information Extractor는 문서에서 자동으로 정보를 추출하여 사용자가 원하는 데이터만 필터링할 수 있도록 돕는 AI 기반 애플리케이션입니다.
Streamlit을 기반으로 제작되었으며, 누구나 쉽게 사용할 수 있습니다.

🔗 웹 애플리케이션 접속

아래 링크를 클릭하면 직접 사용해 볼 수 있습니다.

GitHub Repository

📌 주요 기능

  • GPT 모델 선택: 다양한 GPT 모델 사용 가능 (gpt-3.5-turbo 등)
  • 텍스트 분석: 업로드된 문서에서 주요 정보를 추출
  • 자동 태깅: 키워드를 자동 감지하여 분류
  • 엑셀 파일 지원: 문서 내 데이터를 필터링하고, 특정 정보를 추출하여 엑셀로 제공
  • UI 간편성: Streamlit을 이용한 직관적인 인터페이스 제공

📌 사용 방법

  1. 위의 "Information Extractor 사용하기" 버튼을 클릭하여 웹 애플리케이션에 접속
  2. GPT API Key 입력
  3. GPT 모델 선택 (예: gpt-3.5-turbo)
  4. 프롬프트 문서 (.docx) 및 데이터 엑셀 파일 (.xlsx) 업로드
  5. 추출할 데이터 컬럼 선택 후 실행
  6. 결과 확인 및 다운로드 (필요한 경우)
